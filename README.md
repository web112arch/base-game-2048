# Base Game 2048

Base Game 2048 is a clean and lightweight implementation of the classic 2048 puzzle game.

This first version is **purely web-based**, focusing on gameplay, simplicity, and performance.
Blockchain integration (Base L2) will be added in future versions.

## Features
- Classic 2048 gameplay
- Runs fully in the browser
- No frameworks, no dependencies
- Clean and readable code

## Roadmap
- UI improvements
- Mobile swipe support
- Base L2 on-chain score tracking
- Global leaderboard
